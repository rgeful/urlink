# UrLink

A modern link-in-bio platform that combines clean creator profiles with lightweight social features. Inspired by Linktree, Hoo.be, and X/Instagram, UrLink lets users share their links, customize their profile card, create posts, follow others, and track simple analytics â€” all in a smooth, minimal interface.

## Features

### Customizable Profile
- Profile picture, bio, and social icons
- Add unlimited links with thumbnails
- High-contrast card layout

### Link Management
- Add, edit, and reorder links
- Built-in click tracking
- Clean card-style presentation

### Social Layer
- Create text or image posts
- Like and repost content
- Follow/unfollow users
- Profile feed + home feed

### Dashboard
- Edit profile details
- Manage links
- View basic analytics (profile views, link clicks)

### Public Pages
- SEO-friendly shareable profile at `/u/username`
- Fast, minimal, mobile-first design

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Supabase** (PostgreSQL, Auth, Storage)
- **Prisma ORM**
- **Vercel** (deployment)

## Project Goals

UrLink is built to be:

- Fast and simple to use
- Visually clean and satisfying
- Creator-friendly
- Mobile-optimized
- Easy to share and easy to customize
